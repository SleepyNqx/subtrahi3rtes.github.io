Gleamful DEV

A glowy, animated, glassmorphic Discord theme. Inspired by neon gradients, blur-heavy UI, and motion-rich interactions.

Download Theme GitHub

Animated Gradients

Usernames, links, timestamps, badges, and icons use animated gradients exactly like the theme.

Glassmorphism

Heavy backdrop blur, transparency, and layered shadows across popouts, modals, and sidebars.

Micro‑Animations

Hover scaling, glow pulses, smooth transforms on avatars, emojis, buttons, and media.

Fully Customizable

CSS variables for accent color, blur strength, gradients, fonts, and background image.
